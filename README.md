# 🌿 Clínica Vitalis — Landing Page

Projeto pessoal de uma landing page de conversão para uma clínica fictícia 
de Reabilitação & Medicina Integrada chamada Vitalis.

A ideia foi criar algo que parecesse real — com identidade visual própria, 
foco em conversão via WhatsApp e atenção aos detalhes de UI/UX — 
sem usar nenhum framework ou biblioteca externa.

## ✨ O que tem nesse projeto
- Layout responsivo e mobile-first
- Animações de scroll com IntersectionObserver
- Slider de depoimentos em JS puro
- Counter animation nos números da página
- Formulário que monta mensagem e abre direto no WhatsApp
- Navbar com blur ao rolar
- Botão flutuante de WhatsApp com animação

## 🛠️ Tecnologias

- HTML5 semântico
- CSS puro com custom properties (variáveis CSS)
- JavaScript vanilla

> Sem React, sem Tailwind, sem dependências. Um arquivo só.
